# MCQ Tester v1.0.0

First packaged Windows release of MCQ Tester, a desktop multiple choice practice app built with PySide6.

## Download

- Recommended: `MCQ_Tester_Setup_v1.0.0.exe`
- Portable option: `MCQ_Tester_v1.0.0_windows.zip`

## What's included

- Study mode with one-question-at-a-time feedback, answer highlighting, and explanations.
- Test mode with one-question-at-a-time navigation, flagging, answer status indicators, and final grading.
- Full page test mode with all questions on one scrollable page, flagging, unanswered/flagged counters, and final grading.
- JSON paste/import workflow for Custom GPT-generated questions.
- Automatic cleanup and validation for common GPT JSON formatting issues.
- Save/load support for reusable local test files.
- App icon, Start menu shortcut, optional desktop shortcut, and uninstaller in the Windows installer.

## Notes

- Saved tests are stored in the local `saved_tests` folder when running from source.
- In the packaged app, saved tests are stored under the user's local app data folder.
- The installer does not require administrator privileges.
